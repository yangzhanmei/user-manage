## 简介
该系统是一个人员管理的小系统，可以对人员进行增删改查
### 技术栈
react、redux、express、webpack、node.js、mysql、mocha
### 运行方式
git clone https://github.com/yangzhanmei/user-manage

cd user-manage

npm run build

npm start

打开端口http://localhost:3000/
### 测试
npm test
